# [PT-BR]Template de tela de Cadastro

### Objetivo Principal:

Este é um template de tela de cadastro tendo como objetivo principal o desenvolvimento de habilidades com HTML, CSS e Javascript

### Principais Funcionalidades

O programa possui funcionalidades básicas de uma tela de cadastro comum, sendo elas:
- Inputs personalizados
- Máscaras de input (ex: telefone, cpf, etc...)
- Verificação de email válido
- Confirmação se emails e senhas são iguais
- Verificação se a senha atende aos requisitos mínimos de uma senha forte

### Funcionalidade bônus

O programa também utiliza Javascript para consumir uma API na função de buscar endereço pelo CEP informado.

# [ENG]Registration Screen Template
### Main Objective:
This is a registration screen template with the main objective of developing skills in HTML, CSS, and Javascript.

### Main Features
The program has the basic functionalities of a common registration screen, which are:

- Customizable inputs
- Input masks (e.g., phone number, CPF, etc.)
- Valid email verification
- Confirmation that emails and passwords are the same
- Verification that the password meets the minimum requirements of a strong password

### Bonus Feature
The program also uses Javascript to consume an API in the function of searching for an address by the entered ZIP code.
